## 图示布局

### 上半部分：注释类型（y 80–180 蓝框卡片）

卡片内列出 4 种注释示例，每行右侧配小标签注明使用场景：

- `// 单行注释` —— 解释这一行
- `/* 块注释 */` —— 跨行说明
- `/// 文档注释` —— 生成 API 文档
- `//! 模块注释` —— 写在文件顶部

### 下半部分：运算符分类（y 220–340，三个并排卡片）

1. 绿框 **算术与比较**：`+ - * / %`、`== != < > <= >=`
2. 蓝框 **逻辑与位运算**：`&& || !`、`& | ^ << >>`
3. 粉框 **特殊运算符**：`.. ..=`、`? . ::`
